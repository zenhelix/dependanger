== Ключевые принципы

Из xref:constitution.adoc[Constitution]:

* **M1**: Зависимости направлены от периферии к ядру
* **M2**: Доменная модель изолирована от инфраструктуры
* **E1**: Расширяемость через SPI, не через наследование
* **T1**: Все зависимости инъектируются через конструктор
* **C1**: Explicit API mode везде

== Ключевые сущности

Из xref:domain-model.adoc[Доменной модели]:

* **Library** — зависимость с Maven координатами, тегами, requires
* **Bundle** — именованная группа библиотек
* **BomImport** — импорт версий из внешнего Maven BOM
* **TargetPlatform** — целевая платформа с JDK/Kotlin constraints
* **Distribution** — профиль генерации с фильтрацией
* **Settings** — глобальные настройки валидации и генерации
