== Обзор раздела

Этот раздел описывает все точки входа в систему Dependanger:
Gradle Plugin для интеграции в билд, CLI для standalone использования, и программный API для интеграции с другими инструментами.

== Выбор интерфейса

=== Используйте Gradle Plugin если:

* Проект уже использует Gradle
* Нужна интеграция с Gradle lifecycle
* Хотите Kotlin DSL для конфигурации

=== Используйте CLI если:

* Нужен standalone инструмент
* Интеграция в CI/CD pipeline
* Проект не использует Gradle

=== Используйте Library API если:

* Создаете свой инструмент поверх Dependanger
* Нужен программный доступ к функциональности

== Как добавить описание интерфейса

. Скопируйте шаблон: `docs/_templates/interface-template.adoc`
. Заполните все секции
. Добавьте в таблицу выше
