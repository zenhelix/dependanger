[[overview]]
== Обзор

Модуль `dependanger-license` проверяет лицензии библиотек и их соответствие политикам организации (whitelist/blacklist).
Информация о лицензиях получается из нескольких источников по приоритету: User-defined > Maven POM > ClearlyDefined API.
Результаты хранятся в `EffectiveMetadata.extensions` через `LicenseViolationsExtensionKey`.

Приоритет: **P3** (желательно).

[[spec-license-check]]
== SPEC-11.1: LicenseCheckProcessor.process()

[[purpose-license]]
=== Назначение

Собрать все библиотеки из EffectiveMetadata, для каждой определить лицензии (одну или несколько — dual-license) из нескольких источников по приоритету (DECLARED > MAVEN_POM > CLEARLY_DEFINED), классифицировать по категориям, проверить соответствие whitelist/blacklist политикам с учётом dual-license семантики (настраиваемой через `dualLicensePolicy: OR | AND`) и сохранить нарушения в extensions.

[[algorithm-license]]
=== Алгоритм

[source,text]
----
1. Получить настройки licenseCheck из context.settings
2. Собрать библиотеки для проверки:
   a. Все библиотеки из metadata.libraries с version != null
   b. Исключить библиотеки из ignoreLibraries
   c. Если includeTransitives=true -> добавить транзитивные из extensions (FLAT_DEPENDENCIES)
3. Создать LicenseResolver с настроенными источниками и кешем
4. Для каждой библиотеки (параллельно, PARALLEL_IO):
   a. Проверить user-defined лицензию в context (если есть -> LicenseSource.DECLARED)
   b. Если нет -> вызвать licenseResolver.resolve(group, artifact, version)
   c. Результат: List<LicenseResult> -- одна или несколько лицензий (dual-license)
5. Для каждого List<LicenseResult> проверить политики (семантика зависит от dualLicensePolicy):
   ЕСЛИ dualLicensePolicy == OR (дефолт, dual-license = выбор одной из альтернатив):
   a. Если deniedLicenses не пуст:
      - Violation ТОЛЬКО если ВСЕ лицензии из списка в denied (нет допустимой альтернативы)
      - Если хотя бы одна НЕ в denied -> OK
   b. Если allowedLicenses не пуст:
      - OK если ХОТЯ БЫ ОДНА лицензия в allowed
      - Violation если НИ ОДНА не в allowed
   ЕСЛИ dualLicensePolicy == AND (dual-license = обязательства по всем лицензиям):
   a. Если deniedLicenses не пуст:
      - Violation если ЛЮБАЯ лицензия в denied
   b. Если allowedLicenses не пуст:
      - OK только если ВСЕ лицензии в allowed
      - Violation если хотя бы одна НЕ в allowed
   ОБЩЕЕ (не зависит от policy):
   c. Если warnOnCopyleft и ЛЮБАЯ category in (WEAK_COPYLEFT, STRONG_COPYLEFT) -> WARNING
   d. Если warnOnUnknown и ВСЕ category == UNKNOWN -> WARNING
6. Собрать все LicenseViolation
7. Записать в metadata:
   a. withExtension(LicenseViolationsExtensionKey, violations)
8. Добавить diagnostics:
   a. INFO: "License check completed: X libraries, Y violations"
   b. Если failOnDenied && есть DENIED violations -> ERROR
   c. Если failOnUnknown && есть UNKNOWN лицензии -> ERROR
   d. Если failOnCopyleft && есть copyleft -> ERROR
----

[[spec-license-resolver]]
== SPEC-11.2: LicenseResolver.resolve()

[[algorithm-resolver]]
=== Алгоритм

Возвращает `List<LicenseResult>` -- одну или несколько лицензий для библиотеки.
Множественные лицензии возникают при dual-license (POM с несколькими `<license>`) или SPDX expression ("MIT OR Apache-2.0").

[source,text]
----
1. Проверить кеш для group:artifact:version
2. Если есть в кеше и не expired -> вернуть из кеша
3. Последовательно по источникам (в порядке приоритета):
   a. DECLARED:
      - Проверить контекст на user-defined лицензию для данной библиотеки
      - Если есть -> нормализовать SPDX ID, определить категорию
      - Вернуть listOf(LicenseResult(source=DECLARED))
   b. MAVEN_POM:
      - Скачать POM файл через MavenPomDownloader (из dependanger-maven-resolver)
      - Распарсить секцию <licenses>
      - Для КАЖДОЙ <license> (не только первой):
        i.   Извлечь <name> и <url>
        ii.  Нормализовать name -> SPDX ID через SpdxLicenseMapper
        iii. Определить LicenseCategory по SPDX ID
        iv.  Создать LicenseResult(source=MAVEN_POM)
      - Если найдены лицензии -> вернуть List<LicenseResult>
   c. CLEARLY_DEFINED:
      - GET https://api.clearlydefined.io/definitions/maven/mavencentral/{group}/{artifact}/{version}
      - Парсить response -> licensed.declared (SPDX expression)
      - Если expression содержит "OR" -> split и создать LicenseResult для каждой
      - Если одна лицензия -> listOf(LicenseResult(source=CLEARLY_DEFINED))
      - Вернуть List<LicenseResult>
4. Если ни один источник не дал результата:
   - Вернуть listOf(LicenseResult(spdxId=null, licenseName=null, source=UNKNOWN, category=UNKNOWN))
5. Сохранить результат в кеш
6. Вернуть List<LicenseResult>
----

----
public class LicenseCheckProcessor(
    private val licenseResolver: LicenseResolver = LicenseResolver(),
) : EffectiveMetadataProcessor {
    override val id: String = "license-check"
    override val phase: ProcessingPhase = ProcessingPhase.LICENSE_CHECK  // order=130, PARALLEL_IO
    override val isOptional: Boolean = true

    override suspend fun process(
        metadata: EffectiveMetadata,
        context: ProcessingContext,
    ): EffectiveMetadata {
        val settings = context.settings.licenseCheck ?: return metadata
        val libraries = collectLibraries(metadata, settings)

        if (libraries.isEmpty()) {
            return metadata.withDiagnostic(
                Diagnostics.info("LICENSE_NO_LIBS", "No libraries to check")
            )
        }

        // Параллельный резолвинг лицензий (каждая библиотека -> List<LicenseResult>)
        val results: List<Pair<EffectiveLibrary, List<LicenseResult>>> = coroutineScope {
            libraries.map { lib ->
                async {
                    val declared = context.getDeclaredLicense(lib.alias)
                    val licenses = if (declared != null) {
                        listOf(resolveDeclared(declared))
                    } else {
                        licenseResolver.resolve(lib.group, lib.artifact, lib.version!!.value)
                    }
                    lib to licenses
                }
            }.awaitAll()
        }

        // Проверить политики (семантика определяется settings.dualLicensePolicy)
        val violations = mutableListOf<LicenseViolation>()
        val diagnostics = mutableListOf<Diagnostics>()

        for ((lib, licenses) in results) {
            checkPolicies(lib, licenses, settings, violations, diagnostics)
        }

        // Записать результаты
        var updatedMetadata = metadata
            .withExtension(LicenseViolationsExtensionKey, violations)

        for (diag in diagnostics) {
            updatedMetadata = updatedMetadata.withDiagnostic(diag)
        }

        updatedMetadata = updatedMetadata.withDiagnostic(
            Diagnostics.info(
                "LICENSE_CHECK_COMPLETE",
                "License check completed: ${results.size} libraries, ${violations.size} violations"
            )
        )

        if (settings.failOnDenied && violations.any { it.message.contains("denied") }) {
            updatedMetadata = updatedMetadata.withDiagnostic(
                Diagnostics.error("LICENSE_DENIED_FOUND", "Denied licenses found in ${violations.size} libraries")
            )
        }

        if (settings.failOnUnknown && results.any { (_, licenses) ->
                licenses.all { it.category == LicenseCategory.UNKNOWN }
            }) {
            updatedMetadata = updatedMetadata.withDiagnostic(
                Diagnostics.error("LICENSE_UNKNOWN_FOUND", "Unknown licenses found")
            )
        }

        if (settings.failOnCopyleft && results.any { (_, licenses) ->
                licenses.any { it.category in listOf(LicenseCategory.WEAK_COPYLEFT, LicenseCategory.STRONG_COPYLEFT) }
            }) {
            updatedMetadata = updatedMetadata.withDiagnostic(
                Diagnostics.error("LICENSE_COPYLEFT_FOUND", "Copyleft licenses found")
            )
        }

        return updatedMetadata
    }
}

==== LicenseResolver

[source,kotlin]

public class LicenseResolver(
    private val httpClient: HttpClient = createDefaultClient(),
    private val pomDownloader: MavenPomDownloader? = null,
    private val spdxMapper: SpdxLicenseMapper = SpdxLicenseMapper(),
    private val cache: LicenseCache = LicenseCache(),
    private val timeout: Long = 30_000,
    private val retries: Int = 3,
) {
    /**
     * Резолвит лицензии для библиотеки.
     * Возвращает List<LicenseResult> -- одна или несколько лицензий (dual-license).
     * При dual-license ("MIT OR Apache-2.0") возвращает по элементу на каждую альтернативу.
     */
    public suspend fun resolve(
        group: String,
        artifact: String,
        version: String,
    ): List<LicenseResult> {
        // 1. Проверить кеш
        cache.get(group, artifact, version)?.let { return it }

        // 2. Maven POM (может вернуть несколько <license>)
        val pomResults = tryResolveMavenPom(group, artifact, version)
        if (pomResults.isNotEmpty()) {
            cache.put(group, artifact, version, pomResults)
            return pomResults
        }

        // 3. ClearlyDefined API (может содержать SPDX expression с OR)
        val cdResults = tryResolveClearlyDefined(group, artifact, version)
        if (cdResults.isNotEmpty()) {
            cache.put(group, artifact, version, cdResults)
            return cdResults
        }

        // 4. Неизвестная лицензия
        val unknownResults = listOf(LicenseResult(
            spdxId = null,
            licenseName = null,
            source = LicenseSource.UNKNOWN,
            category = LicenseCategory.UNKNOWN,
        ))
        cache.put(group, artifact, version, unknownResults)
        return unknownResults
    }
}

==== Maven POM парсинг лицензий

[source,kotlin]

private suspend fun tryResolveMavenPom(
    group: String,
    artifact: String,
    version: String,
): List<LicenseResult> {
    return try {
        val pomXml = pomDownloader?.download(group, artifact, version) ?: return emptyList()
        val pomLicenses = parseLicensesFromPom(pomXml)

        if (pomLicenses.isEmpty()) return emptyList()

        // Все <license> элементы из POM -> List<LicenseResult>
        pomLicenses.map { pomLicense ->
            val spdxId = spdxMapper.normalize(pomLicense.name)
            val category = spdxMapper.categorize(spdxId)
            LicenseResult(
                spdxId = spdxId,
                licenseName = pomLicense.name,
                source = LicenseSource.MAVEN_POM,
                category = category,
            )
        }
    } catch (e: Exception) {
        logger.warn(e) { "Failed to resolve license from Maven POM for $group:$artifact:$version" }
        emptyList()
    }
}

private fun parseLicensesFromPom(pomXml: String): List<PomLicense> {
    val factory = DocumentBuilderFactory.newInstance()
    factory.setFeature("http://apache.org/xml/features/disallow-doctype-decl", true)
    val builder = factory.newDocumentBuilder()
    val doc = builder.parse(InputSource(StringReader(pomXml)))

    val licensesNode = doc.getElementsByTagName("licenses").item(0) ?: return emptyList()
    val licenseNodes = (licensesNode as Element).getElementsByTagName("license")

    return (0 until licenseNodes.length).map { i ->
        val node = licenseNodes.item(i) as Element
        PomLicense(
            name = node.getElementsByTagName("name").item(0)?.textContent?.trim() ?: "",
            url = node.getElementsByTagName("url").item(0)?.textContent?.trim(),
        )
    }
}

private suspend fun tryResolveClearlyDefined(
    group: String,
    artifact: String,
    version: String,
): List<LicenseResult> {
    return try {
        val response = executeWithRetry {
            httpClient.get("https://api.clearlydefined.io/definitions/maven/mavencentral/$group/$artifact/$version") {
                timeout { requestTimeoutMillis = this@LicenseResolver.timeout }
            }
        }

        if (response.status != HttpStatusCode.OK) return emptyList()

        val cdResponse = response.body<ClearlyDefinedResponse>()
        val declared = cdResponse.licensed?.declared ?: return emptyList()

        if (declared.isBlank() || declared == "NOASSERTION") return emptyList()

        // Парсинг SPDX expression: "MIT OR Apache-2.0" -> ["MIT", "Apache-2.0"]
        // Базовый парсер: split по " OR " (top-level OR, без скобок)
        val licenseIds = parseSpdxExpression(declared)

        licenseIds.map { rawId ->
            val spdxId = spdxMapper.normalize(rawId.trim())
            val category = spdxMapper.categorize(spdxId)
            LicenseResult(
                spdxId = spdxId,
                licenseName = rawId.trim(),
                source = LicenseSource.CLEARLY_DEFINED,
                category = category,
            )
        }
    } catch (e: Exception) {
        logger.warn(e) { "Failed to resolve license from ClearlyDefined for $group:$artifact:$version" }
        emptyList()
    }
}

/**
 * Базовый парсер SPDX expressions.
 * Поддерживает: "MIT", "MIT OR Apache-2.0", "GPL-3.0-only OR MIT OR BSD-3-Clause"
 * Split по " OR " (case-insensitive). "AND" не разделяется -- трактуется как одна лицензия.
 * Скобки не поддерживаются (edge case для будущей реализации).
 */
private fun parseSpdxExpression(expression: String): List<String> {
    return expression.split(Regex("\\s+OR\\s+", RegexOption.IGNORE_CASE))
        .map { it.trim() }
        .filter { it.isNotBlank() }
}

public class SpdxLicenseMapper {
    /**
     * Нормализует произвольное название лицензии в SPDX ID.
     * Использует lookup таблицу для распространённых вариаций.
     */
    public fun normalize(licenseName: String): String? {
        // Точное совпадение
        if (licenseName in SPDX_IDS) return licenseName

        // Нормализация через lookup
        val normalized = LICENSE_ALIASES[licenseName.lowercase().trim()]
        if (normalized != null) return normalized

[[policy-check]]
== Проверка политик

=== DualLicensePolicy

Определяет семантику проверки для библиотек с несколькими лицензиями (dual-license).

[source,kotlin]
----
/**
 * Политика проверки dual-license библиотек.
 * Определяет, как интерпретировать наличие нескольких лицензий у одной библиотеки.
 */
@Serializable
public enum class DualLicensePolicy {
    /**
     * OR-семантика (дефолт): пользователь может выбрать любую из предложенных лицензий.
     * Типичный сценарий: "MIT OR Apache-2.0" — достаточно соблюдать одну.
     * - Denied: violation только если ВСЕ лицензии в denied (нет допустимой альтернативы)
     * - Allowed: OK если хотя бы одна лицензия в allowed
     */
    OR,

    /**
     * AND-семантика: все лицензии применяются одновременно (обязательства по каждой).
     * Типичный сценарий: "GPL-2.0 WITH Classpath-exception-2.0" или проект,
     * требующий соблюдения всех лицензий одновременно.
     * - Denied: violation если ЛЮБАЯ лицензия в denied
     * - Allowed: OK только если ВСЕ лицензии в allowed
     */
    AND,
}
----

NOTE: `DualLicensePolicy` определяется в `dependanger-core` (пакет `io.github.zenhelix.dependanger.core.model`) и используется в `LicenseCheckSettings`.

=== Алгоритм проверки (с конфигурируемой dual-license семантикой)

[source,kotlin]
----
/**
 * Проверяет список лицензий библиотеки против политик.
 * Для dual-license (несколько LicenseResult) семантика определяется dualLicensePolicy:
 * - OR: пользователь выбирает одну из альтернатив (дефолт)
 * - AND: обязательства по всем лицензиям одновременно
 * - Copyleft: warning если ЛЮБАЯ лицензия copyleft (не зависит от policy)
 * - Unknown: warning если ВСЕ лицензии unknown (не зависит от policy)
 */
private fun checkPolicies(
    lib: EffectiveLibrary,
    licenses: List<LicenseResult>,
    settings: LicenseCheckSettings,
    violations: MutableList<LicenseViolation>,
    diagnostics: MutableList<Diagnostics>,
) {
    val spdxIds = licenses.mapNotNull { it.spdxId }
    val separator = if (settings.dualLicensePolicy == DualLicensePolicy.AND) " AND " else " OR "
    val licenseExpression = spdxIds.joinToString(separator).ifEmpty { "UNKNOWN" }

    // 1. Denied list check (семантика зависит от policy)
    if (settings.deniedLicenses.isNotEmpty() && spdxIds.isNotEmpty()) {
        val hasDeniedViolation = when (settings.dualLicensePolicy) {
            DualLicensePolicy.OR -> spdxIds.all { it in settings.deniedLicenses }   // ВСЕ denied -> нет альтернативы
            DualLicensePolicy.AND -> spdxIds.any { it in settings.deniedLicenses }  // ЛЮБАЯ denied -> обязательства
        }
        if (hasDeniedViolation) {
            val message = when (settings.dualLicensePolicy) {
                DualLicensePolicy.OR -> "All licenses are in denied list (no permitted alternative): $licenseExpression"
                DualLicensePolicy.AND -> "License in denied list (AND-policy, all apply): $licenseExpression"
            }
            violations.add(LicenseViolation(
                alias = lib.alias,
                group = lib.group,
                artifact = lib.artifact,
                detectedLicense = licenseExpression,
                category = licenses.first().category,
                message = message,
            ))
        }
    }

    // 2. Allowed list check (семантика зависит от policy)
    if (settings.allowedLicenses.isNotEmpty() && spdxIds.isNotEmpty()) {
        val isAllowed = when (settings.dualLicensePolicy) {
            DualLicensePolicy.OR -> spdxIds.any { it in settings.allowedLicenses }   // хотя бы одна в allowed
            DualLicensePolicy.AND -> spdxIds.all { it in settings.allowedLicenses }  // ВСЕ должны быть в allowed
        }
        if (!isAllowed) {
            val message = when (settings.dualLicensePolicy) {
                DualLicensePolicy.OR -> "None of the licenses are in allowed list: $licenseExpression"
                DualLicensePolicy.AND -> "Not all licenses are in allowed list (AND-policy): $licenseExpression"
            }
            violations.add(LicenseViolation(
                alias = lib.alias,
                group = lib.group,
                artifact = lib.artifact,
                detectedLicense = licenseExpression,
                category = licenses.first().category,
                message = message,
            ))
        }
    }

    // 3. Copyleft warning (любая из лицензий copyleft — не зависит от policy)
    val copyleftLicenses = licenses.filter {
        it.category in listOf(LicenseCategory.WEAK_COPYLEFT, LicenseCategory.STRONG_COPYLEFT)
    }
    if (settings.warnOnCopyleft && copyleftLicenses.isNotEmpty()) {
        val copyleftIds = copyleftLicenses.mapNotNull { it.spdxId }.joinToString(", ")
        diagnostics.add(Diagnostics.warning(
            "LICENSE_COPYLEFT",
            "Library '${lib.alias}' has copyleft license(s): $copyleftIds"
        ))
    }

    // 4. Unknown warning (все лицензии unknown — не зависит от policy)
    if (settings.warnOnUnknown && licenses.all { it.category == LicenseCategory.UNKNOWN }) {
        diagnostics.add(Diagnostics.warning(
            "LICENSE_UNKNOWN",
            "License for library '${lib.alias}' (${lib.group}:${lib.artifact}:${lib.version?.value}) is unknown"
        ))
    }
}
----

[[cache-impl]]
== Кеш лицензий

=== Структура

[source,text]
----
~/.dependanger/cache/licenses/
  org.jetbrains.kotlin/
    kotlin-stdlib/
      2.1.20.json    # LicenseResult сериализованный
  com.fasterxml.jackson.core/
    jackson-databind/
      2.17.0.json
----

=== TTL

- Default: 7 дней (лицензии меняются редко)
- Ключ кеша: `{group}/{artifact}/{version}`
- При недоступности сети: используется стейл кеш с WARNING

=== Реализация

[source,kotlin]
----
internal class LicenseCache(
    private val cacheDir: Path = Path(System.getProperty("user.home"), ".dependanger", "cache", "licenses"),
    private val ttl: Duration = 7.days,
    private val json: Json = Json { prettyPrint = true; ignoreUnknownKeys = true },
) {
    suspend fun get(group: String, artifact: String, version: String): List<LicenseResult>? {
        val file = cacheFile(group, artifact, version)
        if (!file.exists()) return null

        val age = Clock.System.now() - Instant.fromEpochMilliseconds(file.lastModified())
        if (age > ttl) return null

        return try {
            json.decodeFromString<List<LicenseResult>>(file.readText())
        } catch (e: Exception) {
            logger.warn(e) { "Failed to read license cache for $group:$artifact:$version" }
            null
        }
    }

    suspend fun put(group: String, artifact: String, version: String, results: List<LicenseResult>) {
        val file = cacheFile(group, artifact, version)
        file.parentFile.mkdirs()
        file.writeText(json.encodeToString(results))
    }

    fun invalidate(group: String, artifact: String, version: String) {
        cacheFile(group, artifact, version).delete()
    }

    fun clear() {
        cacheDir.toFile().deleteRecursively()
    }

    private fun cacheFile(group: String, artifact: String, version: String): File =
        cacheDir.resolve(group).resolve(artifact).resolve("$version.json").toFile()
}
----

[[edge-cases]]
== Edge Cases и обработка ошибок

[cols="1,2,2"]
|===
| Ситуация | Поведение | Диагностический код

| Maven POM недоступен, кеш пуст | Переход к ClearlyDefined, WARNING | `LICENSE_POM_UNAVAILABLE`
| ClearlyDefined недоступен | category=UNKNOWN, WARNING | `LICENSE_CD_UNAVAILABLE`
| Оба источника недоступны, кеш пуст | category=UNKNOWN, WARNING/ERROR | `LICENSE_ALL_SOURCES_FAILED`
| Кеш есть, сеть недоступна | Используется стейл кеш + WARNING | `LICENSE_STALE_CACHE`
| Лицензия не в SPDX | spdxId=null, category=PROPRIETARY | `LICENSE_NON_SPDX`
| Dual-license (MIT OR Apache-2.0) | resolve() возвращает 2 LicenseResult, checkPolicies проверяет по dualLicensePolicy | --
| Dual-license OR + denied | GPL-3.0 OR MIT, denied=[GPL-3.0], policy=OR -> OK (MIT альтернатива не denied) | --
| Dual-license OR + все denied | GPL-3.0 OR AGPL-3.0, обе denied, policy=OR -> violation "All licenses are in denied list" | `LICENSE_ALL_DENIED`
| Dual-license AND + denied | GPL-3.0 AND MIT, denied=[GPL-3.0], policy=AND -> violation (GPL-3.0 denied, обязательства по всем) | --
| Dual-license AND + allowed | GPL-3.0 AND MIT, allowed=[MIT], policy=AND -> violation (не все в allowed) | --
| Dual-license AND + all allowed | GPL-3.0 AND MIT, allowed=[MIT, GPL-3.0], policy=AND -> OK | --
| dualLicensePolicy не задан | Дефолт OR — обратная совместимость | --
| Пустой whitelist | Whitelist отключён, все разрешены кроме denied | --
| Whitelist + denied конфликт | Denied имеет приоритет -> violation | --
| User-defined override | DSL license("MIT") приоритет над POM | --
| Невалидный SPDX ID | WARNING, используется как custom | `LICENSE_INVALID_SPDX`
| POM без секции licenses | Переход к ClearlyDefined | --
| Deprecated SPDX ID | Нормализация к актуальному (GPL-2.0 -> GPL-2.0-only) | --
| Private библиотека | ClearlyDefined не найдёт, category=UNKNOWN | --
| SPDX expression "MIT OR Apache-2.0" | parseSpdxExpression() split по OR -> 2 LicenseResult | --
| SPDX expression "MIT AND Apache-2.0" | AND не split, трактуется как одна лицензия | --
| Пустой каталог | 0 violations, INFO | `LICENSE_NO_LIBS`
| Таймаут ClearlyDefined | Retry 3 раза, затем WARNING | `LICENSE_CD_TIMEOUT`
| Rate limiting (429) | Exponential backoff (1s, 2s, 4s) | `LICENSE_RATE_LIMITED`
| > 500 библиотек | Параллельные запросы (PARALLEL_IO), batch обработка | --
|===
