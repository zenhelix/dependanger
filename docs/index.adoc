=== Правила нумерации

* **Фичи**: `F###` — порядковый номер по приоритету
* **ADR**: `ADR-###` — хронологический порядок
* **Требования**: `FR-###`, `NFR-###` — в рамках документа

=== Ключевые принципы

* **DRY** — не повторять контент, использовать `xref:` для ссылок
* **Согласованность** — терминология едина во всех документах
* **Иерархия** — 00 (обзор) → 01 (концепции) → 02 (детали фич)
