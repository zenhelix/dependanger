== Описание

Report Generation создаёт единый JSON отчёт, объединяющий информацию о:

* Всех библиотеках с их версиями и источниками
* Доступных обновлениях (из F007)
* Проблемах совместимости (из F008)
* Валидационных ошибках и предупреждениях

Отчёт предназначен для машинного парсинга в CI/CD, дашбордах и инструментах анализа.

== User Stories

=== US-001: Единый отчёт для CI

[quote]
____
Как DevOps инженер, я хочу получать единый JSON отчёт о состоянии зависимостей, чтобы интегрировать его в CI/CD pipeline и дашборды.
____

**Acceptance Criteria:**

* JSON формат с предсказуемой схемой
* Включает все библиотеки, обновления, проблемы
* Можно генерировать через CLI и Gradle

== API использование

[source,kotlin]
----
val result = Dependanger.fromDsl {
    // ... DSL config
}
.withUpdateCheck()
.withCompatibilityAnalysis(targetJdk = 17)
.build()

// Генерация отчёта
val report = result.generateReport()

// Запись в файл
result.writeReportTo(Path.of("dependanger-report.json"))
----

== CLI команда

[source,bash]
----
dependanger report [--output=report.json] [--include-updates] [--include-compatibility]
----

== Gradle Task

[source,bash]
----
./gradlew dependangerReport
----

Результат: `build/dependanger/dependanger-report.json`

== Интеграции

=== GitHub Actions

[source,yaml]
----
- name: Generate dependency report
  run: ./gradlew dependangerReport

- name: Upload report
  uses: actions/upload-artifact@v4
  with:
    name: dependency-report
    path: build/dependanger/dependanger-report.json
----
