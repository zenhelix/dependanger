== Соглашения по коду

=== Именование

==== Классы и интерфейсы

[cols="1,2,2",options="header"]
|===
| Тип | Соглашение | Пример

| Interface
| Существительное или прилагательное
| `LibraryFilter`, `Validatable`

| Implementation
| Описательное имя без суффикса `Impl`
| `TagBasedLibraryFilter` (не `LibraryFilterImpl`)

| Abstract class
| Префикс `Abstract` или `Base`
| `AbstractProcessor`, `BaseGenerator`

| Builder
| Суффикс `Builder`
| `LibraryBuilder`, `CatalogBuilder`

| Factory
| Суффикс `Factory`
| `ProcessorFactory`

| Exception
| Суффикс `Exception`
| `ValidationException`, `ResolutionException`
|===

==== Функции

[cols="1,2,2",options="header"]
|===
| Тип | Соглашение | Пример

| Query (возвращает данные)
| `get*`, `find*`, `is*`, `has*`
| `getLibrary()`, `findByTag()`, `isValid()`

| Command (изменяет состояние)
| Глагол в повелительном наклонении
| `validate()`, `resolve()`, `generate()`

| Factory method
| `create*`, `of`, `from`
| `createDefault()`, `of(...)`, `fromJson()`

| Builder method
| `with*`, `without*`
| `withVersion()`, `withoutCache()`
|===

    data class Resolved(val version: Version) : ResolutionResult()

== Паттерны проектирования

=== Builder Pattern

[source,kotlin]
----
// ✅ Fluent builder с DSL
class CatalogBuilder {
    private var name: String = ""
    private val libraries = mutableListOf<Library>()
    
    fun name(value: String) = apply { name = value }
    
    fun library(alias: String, block: LibraryBuilder.() -> Unit) = apply {
        libraries += LibraryBuilder(alias).apply(block).build()
    }
    
    fun build(): Catalog {
        require(name.isNotBlank()) { "Name is required" }
        return Catalog(name, libraries.toList())
    }
}

// Использование
val catalog = CatalogBuilder()
    .name("my-catalog")
    .library("kotlin-stdlib") {
        group = "org.jetbrains.kotlin"
        artifact = "kotlin-stdlib"
        version = "2.1.0"
    }
    .build()
----

=== Chain of Responsibility (для pipeline)

[source,kotlin]
----
// ✅ Процессор в цепочке
interface EffectiveMetadataProcessor {
    val order: Int
    suspend fun process(metadata: EffectiveMetadata): EffectiveMetadata
}

// Pipeline исполнитель
class ProcessingPipeline(
    private val processors: List<EffectiveMetadataProcessor>
) {
    suspend fun execute(initial: EffectiveMetadata): EffectiveMetadata {
        return processors
            .sortedBy { it.order }
            .fold(initial) { metadata, processor ->
                processor.process(metadata)
            }
    }
}
----

== Рекомендации по модулям

=== Core модуль

**Должен содержать:**

* Доменные модели (`Library`, `Catalog`, `Version`, etc.)
* Интерфейсы для SPI
* Базовая валидация
* DSL markers и базовые builders

**Не должен содержать:**

* Реализации, зависящие от внешних систем
* HTTP клиенты
* Конкретные форматы вывода (TOML, BOM)

=== API модуль

**Должен содержать:**

* Facade (`Dependanger`, `DependangerBuilder`)
* Координация между компонентами
* Конфигурация по умолчанию
* ServiceLoader discovery

**Паттерн фасада:**

[source,kotlin]
----
class Dependanger private constructor(
    private val config: DependangerConfig
) {
    suspend fun generate(): GenerationResult { ... }
    suspend fun validate(): ValidationResult { ... }
    suspend fun checkUpdates(): UpdateReport { ... }
    
    companion object {
        fun builder(): DependangerBuilder = DependangerBuilder()
    }
}
----

=== Feature модули

**Каждый feature модуль должен:**

* Реализовывать один SPI интерфейс
* Быть независимым от других features
* Регистрироваться через ServiceLoader или явную конфигурацию

=== Interface модули (CLI, Gradle Plugin)

**Должны:**

* Быть тонким слоем над API
* Транслировать пользовательский ввод в API вызовы
* Форматировать вывод для конкретной платформы

**Не должны:**

* Содержать бизнес-логику
* Напрямую обращаться к core, минуя api

== Паттерны тестирования

=== Unit тесты

[source,kotlin]
----
class LibraryFilterTest {

    private val sut = TagBasedLibraryFilter()

    @Test
    fun `should include library when tag matches`() {
        // Given
        val library = TestLibraries.kotlinStdlib.copy(tags = setOf("core"))
        val filter = FilterSpec(includeTags = setOf("core"))

        // When
        val result = sut.filter(listOf(library), filter)

        // Then
        assertThat(result).containsExactly(library)
    }

    @Test
    fun `should exclude library when no tags match`() {
        // Given
        val library = TestLibraries.kotlinStdlib.copy(tags = setOf("core"))
        val filter = FilterSpec(includeTags = setOf("android"))

        // When
        val result = sut.filter(listOf(library), filter)

        // Then
        assertThat(result).isEmpty()
    }
}
----

=== Integration тесты

[source,kotlin]
----
@IntegrationTest
class BomImportProcessorIT {

    @TempDir
    lateinit var tempDir: Path

    private lateinit var processor: BomImportProcessor
    private lateinit var mockServer: MockWebServer

    @BeforeEach
    fun setup() {
        mockServer = MockWebServer()
        mockServer.start()

        processor = BomImportProcessor(
            httpClient = HttpClient(MockEngine),
            cache = FileBasedCache(tempDir)
        )
    }

    @Test
    fun `should import versions from Spring Boot BOM`() = runTest {
        // Given
        mockServer.enqueue(MockResponse().setBody(SPRING_BOOT_BOM_XML))

        val metadata = EffectiveMetadata(
            libraries = listOf(
                Library("spring-web", "org.springframework:spring-web")
            ),
            bomImports = listOf(
                BomImport("spring-boot", "org.springframework.boot:spring-boot-dependencies:3.4.0")
            )
        )

        // When
        val result = processor.process(metadata, ProcessingContext())

        // Then
        assertThat(result.libraries.first().version?.value).isEqualTo("6.2.0")
        assertThat(result.libraries.first().versionSource).isEqualTo(VersionSource.BOM_IMPORT)
    }
}
----

=== Functional тесты (E2E)

[source,kotlin]
----
@FunctionalTest
class TomlGenerationFT {

    @TempDir
    lateinit var projectDir: Path

    @Test
    fun `should generate valid TOML from DSL`() {
        // Given
        val dslFile = projectDir.resolve("dependencies.main.kts")
        dslFile.writeText("""
            @file:DependsOn("io.github.zenhelix:dependanger-dsl:0.1.0")

            import io.github.zenhelix.dependanger.dsl.*

            dependanger {
                versions {
                    version("kotlin", "2.1.0")
                }
                libraries {
                    library("kotlin-stdlib", "org.jetbrains.kotlin:kotlin-stdlib") {
                        version = versionRef("kotlin")
                    }
                }
            }
        """.trimIndent())

        // When
        val result = ProcessBuilder("kotlin", dslFile.toString())
            .directory(projectDir.toFile())
            .start()
            .waitFor()

        // Then
        assertThat(result).isEqualTo(0)

        val tomlFile = projectDir.resolve("libs.versions.toml")
        assertThat(tomlFile).exists()
        assertThat(tomlFile.readText()).contains("[versions]")
        assertThat(tomlFile.readText()).contains("kotlin = \"2.1.0\"")
    }
}
