[[overview]]
== Обзор

Модуль `dependanger-report` генерирует единый отчёт о состоянии зависимостей, объединяя данные из EffectiveMetadata и extensions (updates, vulnerabilities, licenses, compatibility, transitives).
Поддерживает форматы: JSON, YAML, Markdown, HTML.

NOTE: `ReportGenerator` -- это **standalone утилита**, а НЕ `EffectiveMetadataProcessor`.
Он не участвует в pipeline и не регистрируется через ServiceLoader.
Вызывается напрямую из CLI (`report` command), Gradle (`dependangerReport` task) и API после завершения pipeline.
В `ProcessingPhase` нет фазы REPORT.

Приоритет: **P3** (желательно).

[[spec-report-generator]]
== SPEC-09.1: ReportGenerator.generate()

[[purpose-report]]
=== Назначение

Собрать данные из `EffectiveMetadata` (библиотеки, плагины, бандлы) и extensions (updates, vulnerabilities, licenses, compatibility, transitives), сформировать отчёт в запрошенном формате.

[[algorithm-report]]
=== Алгоритм

[source,text]
----
1. Построить ReportData из EffectiveMetadata:
   a. summary -> подсчитать счётчики
   b. libraries -> из metadata.libraries
   c. plugins -> из metadata.plugins
   d. bundles -> из metadata.bundles
   e. updates -> из metadata.updates (extension)
   f. compatibility -> из metadata.compatibilityIssues (extension)
   g. vulnerabilities -> из metadata.vulnerabilities (extension)
   h. deprecated -> отфильтровать библиотеки с isDeprecated=true
   i. licenses -> из extensions (LicenseResult для каждой библиотеки)
   j. transitives -> из metadata.transitives (extension)
   k. constraints -> из originalMetadata?.constraints (если передан)
   l. validation -> из metadata.diagnostics (errors + warnings)
2. Отфильтровать секции по списку `sections`
3. Построить summary:
   - libraryCount, pluginCount, bundleCount
   - deprecatedCount -> count isDeprecated
   - updatesAvailable -> updates.size
   - vulnerabilitiesCount, criticalVulnerabilities, highVulnerabilities
   - deniedLicenses, unknownLicenses
   - compatibilityIssues
   - validationErrors, validationWarnings
4. Сериализовать в формат:
   a. JSON -> kotlinx-serialization Json.encodeToString с prettyPrint
   b. YAML -> kaml (kotlinx-serialization YAML): Yaml.default.encodeToString(data)
   c. Markdown -> шаблонный рендеринг (buildString, все 13 секций)
   d. HTML -> Markdown -> commonmark-java Parser+HtmlRenderer -> HTML template с CSS
5. Если outputDir указан -> записать файл (atomic write)
6. Вернуть DependangerReport(format, content, outputPath)
----

==== ReportGenerator.generate()

[source,kotlin]
----
public fun generate(
    effective: EffectiveMetadata,
    settings: ReportSettings = ReportSettings(),
    originalMetadata: DependangerMetadata? = null,
): DependangerReport {
    val format = settings.format
    val sections = settings.sections
    val outputDir = settings.outputDir

    // 1. Собрать данные
    val reportData = buildReportData(effective, sections)

    // 2. Сериализовать в формат
    val content = when (format) {
        ReportFormat.JSON -> renderJson(reportData)
        ReportFormat.YAML -> renderYaml(reportData)
        ReportFormat.MARKDOWN -> renderMarkdown(reportData)
        ReportFormat.HTML -> renderHtml(reportData)
    }

    // 3. Записать файл
    val extension = when (format) {
        ReportFormat.JSON -> "json"
        ReportFormat.YAML -> "yaml"
        ReportFormat.MARKDOWN -> "md"
        ReportFormat.HTML -> "html"
    }
    val outputPath = "$outputDir/dependanger-report.$extension"
    val dir = File(outputDir)
    if (!dir.exists()) dir.mkdirs()

    // Atomic write
    val tempFile = File.createTempFile("report-", ".$extension", dir)
    tempFile.writeText(content)
    val targetFile = File(outputPath)
    tempFile.renameTo(targetFile)

    return DependangerReport(format, content, outputPath)
}

private fun renderMarkdown(data: ReportData): String = buildString {
    appendLine("# Dependanger Report")
    appendLine()
    appendLine("Generated: ${data.generatedAt}")
    data.distribution?.let { appendLine("Distribution: $it") }
    appendLine()

    // Summary (всегда)
    appendLine("## Summary")
    appendLine()
    appendLine("| Metric | Count |")
    appendLine("|--------|-------|")
    appendLine("| Libraries | ${data.summary.libraryCount} |")
    appendLine("| Plugins | ${data.summary.pluginCount} |")
    appendLine("| Bundles | ${data.summary.bundleCount} |")
    if (data.summary.deprecatedCount > 0)
        appendLine("| Deprecated | ${data.summary.deprecatedCount} |")
    if (data.summary.updatesAvailable > 0)
        appendLine("| Updates Available | ${data.summary.updatesAvailable} |")
    if (data.summary.vulnerabilitiesCount > 0) {
        appendLine("| Vulnerabilities | ${data.summary.vulnerabilitiesCount} |")
        appendLine("| Critical | ${data.summary.criticalVulnerabilities} |")
        appendLine("| High | ${data.summary.highVulnerabilities} |")
    }
    if (data.summary.deniedLicenses > 0)
        appendLine("| Denied Licenses | ${data.summary.deniedLicenses} |")
    if (data.summary.compatibilityIssues > 0)
        appendLine("| Compatibility Issues | ${data.summary.compatibilityIssues} |")
    if (data.summary.validationErrors > 0)
        appendLine("| Validation Errors | ${data.summary.validationErrors} |")
    if (data.summary.validationWarnings > 0)
        appendLine("| Validation Warnings | ${data.summary.validationWarnings} |")

    // Libraries
    data.libraries?.let { libs ->
        appendLine()
        appendLine("## Libraries (${libs.size})")
        appendLine()
        appendLine("| Alias | Coordinates | Version | Tags |")
        appendLine("|-------|------------|---------|------|")
        for (lib in libs) {
            val tags = lib.tags.joinToString(", ").ifEmpty { "-" }
            appendLine("| ${lib.alias} | ${lib.group}:${lib.artifact} | ${lib.version ?: "N/A"} | $tags |")
        }
    }

    // Plugins
    data.plugins?.let { plugins ->
        appendLine()
        appendLine("## Plugins (${plugins.size})")
        appendLine()
        appendLine("| Alias | Plugin ID | Version |")
        appendLine("|-------|-----------|---------|")
        for (p in plugins) {
            appendLine("| ${p.alias} | ${p.id} | ${p.version ?: "N/A"} |")
        }
    }

    // Bundles
    data.bundles?.let { bundles ->
        appendLine()
        appendLine("## Bundles (${bundles.size})")
        appendLine()
        for (b in bundles) {
            appendLine("- **${b.alias}**: ${b.libraries.joinToString(", ")}")
        }
    }

    // Versions
    data.versions?.let { versions ->
        appendLine()
        appendLine("## Versions (${versions.size})")
        appendLine()
        appendLine("| Alias | Value | Fallback |")
        appendLine("|-------|-------|----------|")
        for (v in versions) {
            val fallback = if (v.hasFallback) "Yes" else "-"
            appendLine("| ${v.alias} | ${v.value} | $fallback |")
        }
    }

    // Deprecated
    data.deprecated?.let { deprecated ->
        appendLine()
        appendLine("## Deprecated Libraries (${deprecated.size})")
        appendLine()
        appendLine("| Library | Reason | Replacement |")
        appendLine("|---------|--------|-------------|")
        for (d in deprecated) {
            appendLine("| ${d.alias} | ${d.reason ?: "-"} | ${d.replacement ?: "-"} |")
        }
    }

    // Updates
    data.updates?.let { updates ->
        appendLine()
        appendLine("## Available Updates (${updates.size})")
        appendLine()
        appendLine("| Library | Current | Available | Type |")
        appendLine("|---------|---------|-----------|------|")
        for (u in updates) {
            appendLine("| ${u.alias} | ${u.currentVersion} | ${u.availableVersion} | ${u.type} |")
        }
    }

    // Compatibility Issues
    data.compatibility?.let { issues ->
        appendLine()
        appendLine("## Compatibility Issues (${issues.size})")
        appendLine()
        appendLine("| Rule | Severity | Message | Suggestion |")
        appendLine("|------|----------|---------|------------|")
        for (issue in issues) {
            appendLine("| ${issue.ruleId} | ${issue.severity} | ${issue.message} | ${issue.suggestion ?: "-"} |")
        }
    }

    // Vulnerabilities
    data.vulnerabilities?.let { vulns ->
        appendLine()
        appendLine("## Vulnerabilities (${vulns.size})")
        appendLine()
        appendLine("| Library | CVE | Severity | CVSS | Fixed In | Summary |")
        appendLine("|---------|-----|----------|------|----------|---------|")
        for (v in vulns) {
            val cve = v.cveId ?: v.id
            val cvss = v.cvssScore?.let { "%.1f".format(it) } ?: "-"
            appendLine("| ${v.library} | $cve | ${v.severity} | $cvss | ${v.fixedVersion ?: "-"} | ${v.summary} |")
        }
    }

    // Licenses
    data.licenses?.let { licenses ->
        appendLine()
        appendLine("## Licenses (${licenses.size})")
        appendLine()
        appendLine("| Library | License | Status |")
        appendLine("|---------|---------|--------|")
        for (l in licenses) {
            appendLine("| ${l.library} | ${l.licenseName} | ${l.status} |")
        }
    }

    // Transitives
    data.transitives?.let { transitives ->
        appendLine()
        appendLine("## Transitive Dependencies")
        appendLine()
        appendLine("- Direct: ${transitives.directCount}")
        appendLine("- Transitive: ${transitives.transitiveCount}")
        appendLine("- Conflicts: ${transitives.conflictCount}")
        if (transitives.conflicts.isNotEmpty()) {
            appendLine()
            appendLine("### Version Conflicts")
            appendLine()
            appendLine("| Artifact | Versions |")
            appendLine("|----------|----------|")
            for (c in transitives.conflicts) {
                appendLine("| ${c.artifact} | ${c.versions.joinToString(" vs ")} |")
            }
        }
    }

    // Constraints
    data.constraints?.let { constraints ->
        appendLine()
        appendLine("## Constraints (${constraints.size})")
        appendLine()
        for (c in constraints) {
            appendLine("- **${c.type}**: ${c.description}")
        }
    }

    // Validation
    data.validation?.let { validation ->
        appendLine()
        appendLine("## Validation")
        appendLine()
        if (validation.errors.isNotEmpty()) {
            appendLine("### Errors (${validation.errors.size})")
            appendLine()
            for (e in validation.errors) { appendLine("- ❌ [${e.code}] ${e.message}") }
        }
        if (validation.warnings.isNotEmpty()) {
            appendLine("### Warnings (${validation.warnings.size})")
            appendLine()
            for (w in validation.warnings) { appendLine("- ⚠️ [${w.code}] ${w.message}") }
        }
        if (validation.errors.isEmpty() && validation.warnings.isEmpty()) {
            appendLine("All checks passed.")
        }
    }
}

==== HTML рендеринг

Используется библиотека https://github.com/commonmark/commonmark-java[commonmark-java] для конвертации Markdown -> HTML.
Результат оборачивается в HTML-шаблон с базовым CSS.

[source,kotlin]
----
import org.commonmark.parser.Parser
import org.commonmark.renderer.html.HtmlRenderer

private val mdParser = Parser.builder().build()
private val htmlRenderer = HtmlRenderer.builder().build()

private fun renderHtml(data: ReportData): String {
    val markdown = renderMarkdown(data)
    val document = mdParser.parse(markdown)
    val body = htmlRenderer.render(document)
    return htmlTemplate(data.generatedAt, body)
}

[[json-schema]]
== JSON Schema (версия 1.0)

Структура полностью описана в документации F009. Ключевые поля:

- `schemaVersion`: "1.0" -- обязательное
- `generatedAt`: ISO 8601 UTC -- обязательное
- `distribution`: String? -- optional
- `summary`: объект со счётчиками -- обязательное
- Остальные секции -- optional (null если процессор не запущен)

[[edge-cases]]
== Edge Cases и обработка ошибок

[cols="1,2,2"]
|===
| Ситуация | Поведение | Обоснование

| Пустой каталог | Валидный отчёт с пустыми массивами, summary с нулями | Корректный сценарий
| Процессор не запущен | Секция = null в JSON | Не путать с "нет данных" и "проверка не выполнена"
| summaryOnly | Только schemaVersion, generatedAt, distribution, summary | Для быстрого просмотра
| Специальные символы | JSON escaping: `"`, `\n`, unicode | Стандартное JSON поведение
| Очень большой каталог (1000+) | Генерация продолжается, WARNING при > 500KB | Рекомендуется --summary-only
| Нет прав на запись | ERROR "Cannot write report" | Fail-fast
| Файл уже существует | Перезаписать (atomic write через temp) | Идемпотентность
| Невалидный формат | Невозможно: ReportFormat -- sealed enum | Compile-time safety
| Timestamp | Всегда UTC: "2026-01-27T10:30:00Z" | Consistency
| Прерывание (Ctrl+C) | Частичный файл удаляется (atomic write) | Не оставлять мусор
|===
