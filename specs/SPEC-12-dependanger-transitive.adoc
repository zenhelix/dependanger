[[overview]]
== Обзор

Модуль `dependanger-transitive` резолвит транзитивные зависимости из Maven repositories, строит дерево зависимостей, обнаруживает конфликты версий и применяет constraints (VERSION, EXCLUDE, SUBSTITUTE).
Результаты хранятся в `EffectiveMetadata.extensions` через `TransitivesExtensionKey`, `FlatDependenciesExtensionKey` и `VersionConflictsExtensionKey`.

Приоритет: **P4** (будущее).

[[spec-transitive-resolver]]
== SPEC-12.1: TransitiveResolverProcessor.process()

[[purpose-transitive]]
=== Назначение

Для каждой прямой библиотеки из EffectiveMetadata скачать POM из Maven repository, рекурсивно разрезолвить дерево зависимостей (до maxDepth), обнаружить конфликты версий, применить constraints и предоставить результат следующим процессорам (security-check(120), license-check(130) через `includeTransitives`).

NOTE: `TransitiveResolverProcessor` имеет order=101 (SEQUENTIAL) и выполняется **до** SecurityCheck(120) и LicenseCheck(130), чтобы данные `FLAT_DEPENDENCIES` были доступны при `includeTransitives=true`.

[[algorithm-transitive]]
=== Алгоритм

[source,text]
----
1. Получить настройки transitiveResolution из context.settings
2. Получить repositories из settings (default: Maven Central)
3. Получить constraints из context.originalMetadata.constraints
4. Собрать прямые зависимости:
   a. Все библиотеки из metadata.libraries с version != null
   b. Все библиотеки из metadata.plugins с version != null (если includePlugins=true)
5. Создать session-global счётчики:
   a. sessionSeen: ConcurrentSet<String> — для маркировки дубликатов (shared между ветками)
   b. nodeCounter: AtomicInteger(0) — для подсчёта общего количества узлов (preemptive лимит)
   c. maxTransitives = settings.maxTransitives ?: 10_000 (настраиваемый лимит)
6. Для каждой прямой зависимости (параллельно, PARALLEL_IO):
   a. Проверить nodeCounter >= maxTransitives -> early return (лимит достигнут)
   b. Проверить кеш для group:artifact:version
   c. Если нет в кеше:
      i.   Скачать POM из repository через MavenPomDownloader
      ii.  Распарсить <dependencies> секцию (исключить test, provided scope по умолчанию)
      iii. Для каждой зависимости в POM рекурсивно резолвить (до maxDepth, default=10)
      iv.  Обнаружить циклические зависимости (visited: Set<"group:artifact"> — path-specific, иммутабельный) -> isCycle=true
      v.   Пометить дубликаты (isDuplicate=true) через sessionSeen (session-global, shared между ветками)
      vi.  Инкрементировать nodeCounter
   d. Построить TransitiveTree для библиотеки
   e. Сохранить в кеш
7. Post-processing (после построения деревьев):
   a. Обойти деревья, собрать все узлы с isCycle=true -> эмитить TRANSITIVE_CYCLE_DETECTED для каждого
   b. Если totalNodes > 5000 -> WARNING TRANSITIVE_LARGE_TREE
   c. Если nodeCounter >= maxTransitives -> ERROR TRANSITIVE_MAX_EXCEEDED
8. Обнаружить конфликты версий:
   a. Собрать все артефакты с разными версиями
   b. Для каждого конфликта:
      i.   Применить стратегию HIGHEST (наивысшая версия)
      ii.  Если есть constraint -> использовать версию из constraint
      iii. Создать VersionConflict(requestedVersions, resolvedVersion, resolution)
9. Применить constraints (sealed class Constraint):
   a. VersionConstraintDef -> заменить resolved version в дереве и flat list
   b. Exclude -> удалить из дерева и flat list
   c. Substitute -> заменить артефакт (координаты) в дереве и flat list
10. Построить flat list (FlatDependency для каждого уникального артефакта):
    a. Deduplicate по group:artifact
    b. Записать isDirectDependency, isOptional
11. Записать в metadata:
    a. withExtension(TransitivesExtensionKey, trees: List<TransitiveTree>)
    b. withExtension(FlatDependenciesExtensionKey, flat: List<FlatDependency>)
    c. withExtension(VersionConflictsExtensionKey, conflicts: List<VersionConflict>)
12. Добавить diagnostics:
    a. INFO: "Resolved X direct, Y transitive dependencies, Z conflicts"
    b. Для каждого конфликта -> WARNING с деталями
    c. Если constraint на несуществующую -> WARNING
----

public class TransitiveResolverProcessor(
    private val cache: TransitiveCache = TransitiveCache(),
    private val maxDepth: Int = 10,
) : EffectiveMetadataProcessor {
    override val id: String = "transitive-resolver"
    override val phase: ProcessingPhase = ProcessingPhase.TRANSITIVE_RESOLVER  // order=101, SEQUENTIAL
    override val isOptional: Boolean = true

    override suspend fun process(
        metadata: EffectiveMetadata,
        context: ProcessingContext,
    ): EffectiveMetadata {
        val settings = context.settings.transitiveResolution ?: return metadata
        val repositories = settings.repositories.ifEmpty {
            context.settings.repositories.ifEmpty {
                listOf(MavenRepository(url = "https://repo.maven.apache.org/maven2", name = "Maven Central"))
            }
        }
        // NOTE: MavenPomDownloader создаётся здесь, а не в конструкторе,
        // т.к. требует repositories из context settings (см. SPEC-06)
        val pomDownloader = MavenPomDownloader(
            repositories = repositories,
            httpClient = createHttpClient(),
        )
        val constraints = context.originalMetadata.constraints ?: emptyList()
        val effectiveMaxDepth = settings.maxDepth ?: maxDepth
        val scopes = settings.scopes

        val libraries = metadata.libraries.values
            .filter { it.version != null }

        if (libraries.isEmpty()) {
            return metadata.withDiagnostic(
                Diagnostics.info("TRANSITIVE_NO_LIBS", "No libraries to resolve transitives for")
            )
        }

        // Session-global state (shared между всеми ветками)
        val sessionSeen: MutableSet<String> = ConcurrentHashMap.newKeySet()
        val nodeCounter = AtomicInteger(0)
        val effectiveMaxTransitives = settings.maxTransitives ?: 10_000

        // Параллельный резолвинг
        val trees = coroutineScope {
            libraries.map { lib ->
                async {
                    resolveTree(
                        group = lib.group,
                        artifact = lib.artifact,
                        version = lib.version!!.value,
                        repositories = repositories,
                        depth = 0,
                        maxDepth = effectiveMaxDepth,
                        scopes = scopes,
                        includeOptional = settings.includeOptional,
                        visited = setOf("${lib.group}:${lib.artifact}"),
                        sessionSeen = sessionSeen,
                        nodeCounter = nodeCounter,
                        maxTransitives = effectiveMaxTransitives,
                    )
                }
            }.awaitAll()
        }

        // Post-processing: собрать diagnostics для циклов и лимитов
        val totalNodes = nodeCounter.get()
        var updatedMetadata = metadata

        // Cycle diagnostics (обход деревьев)
        val cycles = collectCycles(trees)
        for (cycle in cycles) {
            updatedMetadata = updatedMetadata.withDiagnostic(
                Diagnostics.warning(
                    "TRANSITIVE_CYCLE_DETECTED",
                    "Cyclic dependency detected: ${cycle.group}:${cycle.artifact}:${cycle.version}"
                )
            )
        }

        // Limit diagnostics
        if (totalNodes > 5000 && totalNodes < effectiveMaxTransitives) {
            updatedMetadata = updatedMetadata.withDiagnostic(
                Diagnostics.warning(
                    "TRANSITIVE_LARGE_TREE",
                    "Large dependency tree: $totalNodes transitive nodes (consider reviewing dependencies)"
                )
            )
        }

        if (totalNodes >= effectiveMaxTransitives) {
            updatedMetadata = updatedMetadata.withDiagnostic(
                Diagnostics.error(
                    "TRANSITIVE_MAX_EXCEEDED",
                    "Transitive dependency limit exceeded: $totalNodes >= $effectiveMaxTransitives, resolution was truncated"
                )
            )
        }

        // Обнаружить конфликты
        val allDependencies = collectAllDependencies(trees)
        val conflicts = detectConflicts(allDependencies, constraints)

        // Применить constraints
        val constrainedTrees = applyConstraints(trees, constraints)

        // Построить flat list
        val flatList = buildFlatList(constrainedTrees, libraries)

        // Записать результаты
        updatedMetadata = updatedMetadata
            .withExtension(TransitivesExtensionKey, constrainedTrees)
            .withExtension(FlatDependenciesExtensionKey, flatList)
            .withExtension(VersionConflictsExtensionKey, conflicts)

        updatedMetadata = updatedMetadata.withDiagnostic(
            Diagnostics.info(
                "TRANSITIVE_RESOLVED",
                "Resolved ${libraries.size} direct, ${flatList.count { !it.isDirectDependency }} transitive dependencies, ${conflicts.size} conflicts"
            )
        )

        for (conflict in conflicts) {
            updatedMetadata = updatedMetadata.withDiagnostic(
                Diagnostics.warning(
                    "TRANSITIVE_CONFLICT",
                    "Version conflict: ${conflict.group}:${conflict.artifact} versions ${conflict.requestedVersions.joinToString(", ")} -> resolved ${conflict.resolvedVersion} (${conflict.resolution})"
                )
            )
        }

        return updatedMetadata
    }

    /**
     * Обход деревьев для сбора циклических узлов (isCycle=true).
     */
    private fun collectCycles(trees: List<TransitiveTree>): List<TransitiveTree> {
        val result = mutableListOf<TransitiveTree>()
        fun traverse(tree: TransitiveTree) {
            if (tree.isCycle) result.add(tree)
            tree.children.forEach { traverse(it) }
        }
        trees.forEach { traverse(it) }
        return result
    }
}

==== Рекурсивный резолвинг дерева

[source,kotlin]
----
/**
 * Рекурсивный резолвинг дерева зависимостей.
 *
 * Три механизма отслеживания:
 * - `visited: Set<String>` — path-specific (иммутабельный), для обнаружения ЦИКЛОВ.
 *   Содержит координаты предков текущего узла. Каждый child получает копию + свою координату.
 *   Если coordinate in visited → цикл (isCycle=true), ветка прерывается.
 * - `sessionSeen: MutableSet<String>` — session-global (ConcurrentHashMap.newKeySet()),
 *   для маркировки ДУБЛИКАТОВ. Shared между всеми параллельными ветками.
 *   Если coordinate уже в sessionSeen → isDuplicate=true (зависимость уже резолвлена в другой ветке).
 * - `nodeCounter: AtomicInteger` — session-global, preemptive лимит количества узлов.
 *   Если nodeCounter >= maxTransitives → early return (предотвращение OOM).
 */
private suspend fun resolveTree(
    group: String,
    artifact: String,
    version: String,
    repositories: List<MavenRepository>,
    depth: Int,
    maxDepth: Int,
    scopes: List<String>,
    includeOptional: Boolean,
    visited: Set<String>,
    sessionSeen: MutableSet<String>,
    nodeCounter: AtomicInteger,
    maxTransitives: Int,
): TransitiveTree {
    val coordinate = "$group:$artifact"

    // Проверить preemptive лимит (предотвращение OOM)
    if (nodeCounter.get() >= maxTransitives) {
        return TransitiveTree(group, artifact, version, children = emptyList())
    }

    // Проверить максимальную глубину
    if (depth >= maxDepth) {
        logger.warn { "Max depth $maxDepth reached for $coordinate:$version, branch truncated" }
        return TransitiveTree(group, artifact, version, children = emptyList())
    }

    // Проверить дубликат (уже резолвлен в другой ветке)
    val isDuplicate = !sessionSeen.add(coordinate) // returns false если уже присутствовал
    if (isDuplicate) {
        // Зависимость уже резолвлена в другой ветке — вернуть из кеша с isDuplicate=true
        val cached = cache.get(group, artifact, version)
        return cached?.copy(isDuplicate = true, children = emptyList())
            ?: TransitiveTree(group, artifact, version, isDuplicate = true, children = emptyList())
    }

    // Инкрементировать счётчик узлов
    nodeCounter.incrementAndGet()

    // Проверить кеш (не дубликат, но может быть в кеше от предыдущей сессии)
    cache.get(group, artifact, version)?.let { cached ->
        return cached.copy(isDuplicate = false)
    }

    // Скачать и распарсить POM
    val pomXml = try {
        downloadPom(group, artifact, version, repositories)
    } catch (e: Exception) {
        logger.warn(e) { "Failed to download POM for $coordinate:$version" }
        return TransitiveTree(group, artifact, version, children = emptyList())
    }

    if (pomXml == null) {
        return TransitiveTree(group, artifact, version, children = emptyList())
    }

    // Парсить зависимости из POM
    val pomDependencies = parsePomDependencies(pomXml, scopes, includeOptional)

    // Рекурсивно резолвить children
    val children = coroutineScope {
        pomDependencies.map { dep ->
            async {
                val childCoordinate = "${dep.group}:${dep.artifact}"
                if (childCoordinate in visited) {
                    // Цикл: зависимость уже в предках текущего пути → isCycle=true (НЕ isDuplicate)
                    TransitiveTree(dep.group, dep.artifact, dep.version, isCycle = true, scope = dep.scope, children = emptyList())
                } else {
                    resolveTree(
                        group = dep.group,
                        artifact = dep.artifact,
                        version = dep.version ?: "UNKNOWN",
                        repositories = repositories,
                        depth = depth + 1,
                        maxDepth = maxDepth,
                        scopes = scopes,
                        includeOptional = includeOptional,
                        visited = visited + childCoordinate,
                        sessionSeen = sessionSeen,
                        nodeCounter = nodeCounter,
                        maxTransitives = maxTransitives,
                    ).copy(scope = dep.scope)
                }
            }
        }.awaitAll()
    }

    val tree = TransitiveTree(group, artifact, version, children = children)

    // Сохранить в кеш
    cache.put(group, artifact, version, tree)

    return tree
}

private fun parsePomDependencies(
    pomXml: String,
    scopes: List<String>,
    includeOptional: Boolean,
): List<PomDependencyEntry> {
    val factory = DocumentBuilderFactory.newInstance()
    factory.setFeature("http://apache.org/xml/features/disallow-doctype-decl", true)
    val builder = factory.newDocumentBuilder()
    val doc = builder.parse(InputSource(StringReader(pomXml)))

    // Собрать properties для подстановки ${property}
    val properties = parseProperties(doc)

    // Собрать managedVersions из <dependencyManagement>
    val managedVersions = parseDependencyManagement(doc, properties)

    // Парсить <dependencies> (только прямые, не из dependencyManagement)
    val dependenciesNode = doc.documentElement
        .getElementsByTagName("dependencies")
        .let { nodes ->
            (0 until nodes.length)
                .map { nodes.item(it) }
                .firstOrNull { it.parentNode == doc.documentElement }
        } ?: return emptyList()

    val depNodes = (dependenciesNode as Element).getElementsByTagName("dependency")
    return (0 until depNodes.length).mapNotNull { i ->
        val node = depNodes.item(i) as Element
        val group = resolveProperty(node.getTagValue("groupId"), properties) ?: return@mapNotNull null
        val artifact = resolveProperty(node.getTagValue("artifactId"), properties) ?: return@mapNotNull null
        val version = resolveProperty(node.getTagValue("version"), properties)
            ?: managedVersions["$group:$artifact"]
        val scope = node.getTagValue("scope") ?: "compile"
        val optional = node.getTagValue("optional")?.toBooleanStrictOrNull() ?: false

        // Фильтр по scope
        if (scope !in scopes) return@mapNotNull null
        if (optional && !includeOptional) return@mapNotNull null

        PomDependencyEntry(group, artifact, version, scope)
    }
}

internal data class PomDependencyEntry(
    val group: String,
    val artifact: String,
    val version: String?,
    val scope: String,
)

private fun resolveProperty(value: String?, properties: Map<String, String>): String? {
    if (value == null) return null
    if (!value.contains("\${")) return value

    val regex = Regex("""\$\{([^}]+)}""")
    return regex.replace(value) { match ->
        val propName = match.groupValues[1]
        properties[propName] ?: match.value
    }
}

private fun detectConflicts(
    allDependencies: Map<String, List<String>>, // group:artifact -> [versions]
    constraints: List<Constraint>,
): List<VersionConflict> {
    return allDependencies
        .filter { (_, versions) -> versions.distinct().size > 1 }
        .map { (coordinate, versions) ->
            val (group, artifact) = coordinate.split(":", limit = 2)
            val distinctVersions = versions.distinct()

            // Проверить constraint
            val versionConstraint = constraints.filterIsInstance<Constraint.VersionConstraintDef>()
                .firstOrNull { it.coordinates == coordinate }

            val resolvedVersion = if (versionConstraint != null) {
                (versionConstraint.version as? VersionReference.Literal)?.value
                    ?: selectHighest(distinctVersions)
            } else {
                selectHighest(distinctVersions)
            }

            val resolution = if (versionConstraint != null) {
                ConflictResolutionStrategy.CONSTRAINT
            } else {
                ConflictResolutionStrategy.HIGHEST
            }

            VersionConflict(
                group = group,
                artifact = artifact,
                requestedVersions = distinctVersions,
                resolvedVersion = resolvedVersion,
                resolution = resolution,
            )
        }
}

/**
 * Выбрать наивысшую версию из списка.
 * Используется семантическое сравнение, с fallback на строковое.
 */
private fun selectHighest(versions: List<String>): String {
    return versions.sortedWith(VersionComparator.comparator()).last()
}

==== Применение constraints

[source,kotlin]

private fun applyConstraints(
    trees: List<TransitiveTree>,
    constraints: List<Constraint>,
): List<TransitiveTree> {
    var result = trees

    for (constraint in constraints) {
        result = when (constraint) {
            is Constraint.VersionConstraintDef -> applyVersionConstraint(result, constraint)
            is Constraint.Exclude -> applyExcludeConstraint(result, constraint)
            is Constraint.Substitute -> applySubstituteConstraint(result, constraint)
        }
    }

    return result
}

private fun applyVersionConstraint(
    trees: List<TransitiveTree>,
    constraint: Constraint.VersionConstraintDef,
): List<TransitiveTree> {
    val (group, artifact) = constraint.coordinates.split(":", limit = 2)
    val version = (constraint.version as? VersionReference.Literal)?.value ?: return trees
    return trees.map { tree ->
        if (tree.group == group && tree.artifact == artifact) {
            tree.copy(version = version)
        } else {
            tree.copy(children = applyVersionConstraint(tree.children, constraint))
        }
    }
}

private fun applyExcludeConstraint(
    trees: List<TransitiveTree>,
    constraint: Constraint.Exclude,
): List<TransitiveTree> {
    return trees
        .filter { !(it.group == constraint.group && it.artifact == constraint.artifact) }
        .map { tree ->
            tree.copy(children = applyExcludeConstraint(tree.children, constraint))
        }
}

private fun applySubstituteConstraint(
    trees: List<TransitiveTree>,
    constraint: Constraint.Substitute,
): List<TransitiveTree> {
    val (fromGroup, fromArtifact) = constraint.from.split(":", limit = 2)
    val toParts = constraint.to.split(":", limit = 3)
    val toGroup = toParts[0]
    val toArtifact = toParts[1]
    val toVersion = toParts.getOrNull(2)
    return trees.map { tree ->
        if (tree.group == fromGroup && tree.artifact == fromArtifact) {
            tree.copy(
                group = toGroup,
                artifact = toArtifact,
                version = toVersion ?: tree.version,
            )
        } else {
            tree.copy(children = applySubstituteConstraint(tree.children, constraint))
        }
    }
}

private fun buildFlatList(
    trees: List<TransitiveTree>,
    directLibraries: Collection<EffectiveLibrary>,
): List<FlatDependency> {
    val directCoordinates = directLibraries.map { "${it.group}:${it.artifact}" }.toSet()
    val seen = mutableMapOf<String, FlatDependency>()

    fun traverse(tree: TransitiveTree) {
        val coordinate = "${tree.group}:${tree.artifact}"
        if (coordinate !in seen && tree.version != null) {
            seen[coordinate] = FlatDependency(
                group = tree.group,
                artifact = tree.artifact,
                version = tree.version,
                isDirectDependency = coordinate in directCoordinates,
                isOptional = false,
            )
        }
        tree.children.forEach { traverse(it) }
    }

    trees.forEach { traverse(it) }
    return seen.values.toList()
}

[[pom-download]]
== Скачивание POM

Модуль использует `MavenPomDownloader` из `dependanger-maven-resolver` для скачивания POM файлов.

=== URL формирование

[source,text]
----
{repositoryUrl}/{group.replace('.','/')}/{artifact}/{version}/{artifact}-{version}.pom
----

=== Пример

[source,text]
----
https://repo.maven.apache.org/maven2/org/springframework/spring-web/6.2.0/spring-web-6.2.0.pom
----

=== Стратегия поиска по repositories

[source,text]
----
1. Для каждого repository в порядке приоритета:
   a. Попытка скачать POM
   b. Если 200 OK -> парсить и вернуть
   c. Если 404 -> перейти к следующему repository
   d. Если 5xx/timeout -> retry 3 раза, затем перейти к следующему
2. Если ни один repository не вернул POM -> WARNING, пустое дерево для библиотеки
----

internal class TransitiveCache(
    private val cacheDir: Path = Path(System.getProperty("user.home"), ".dependanger", "cache", "transitives"),
    private val ttl: Duration = 24.hours,
    private val json: Json = Json {
        prettyPrint = true
        ignoreUnknownKeys = true
        encodeDefaults = false
    },
) {
    suspend fun get(group: String, artifact: String, version: String): TransitiveTree? {
        val file = cacheFile(group, artifact, version)
        if (!file.exists()) return null

        val age = Clock.System.now() - Instant.fromEpochMilliseconds(file.lastModified())
        if (age > ttl) return null

        return try {
            json.decodeFromString<TransitiveTree>(file.readText())
        } catch (e: Exception) {
            logger.warn(e) { "Failed to read transitive cache for $group:$artifact:$version" }
            null
        }
    }

    suspend fun put(group: String, artifact: String, version: String, tree: TransitiveTree) {
        val file = cacheFile(group, artifact, version)
        file.parentFile.mkdirs()
        file.writeText(json.encodeToString(tree))
    }

    fun invalidate(group: String, artifact: String, version: String) {
        cacheFile(group, artifact, version).delete()
    }

    fun clear() {
        cacheDir.toFile().deleteRecursively()
    }

    private fun cacheFile(group: String, artifact: String, version: String): File =
        cacheDir.resolve(group).resolve(artifact).resolve("$version.json").toFile()
}

[cols="1,2,2"]
|===
| Ситуация | Поведение | Диагностический код

| POM не найден (404) | WARNING, пустое дерево для библиотеки | `TRANSITIVE_POM_NOT_FOUND`
| Repository недоступен | Retry с exponential backoff, затем WARNING | `TRANSITIVE_REPO_UNAVAILABLE`
| Циклическая зависимость (A -> B -> C -> A) | Цикл обнаруживается через path-specific `visited` set, ветка прерывается, `isCycle=true`. Post-processing: WARNING TRANSITIVE_CYCLE_DETECTED | `TRANSITIVE_CYCLE_DETECTED`
| Дубликат (A->B->D, A->C->D) | D обнаруживается через session-global `sessionSeen` set, `isDuplicate=true`, children из кеша. Семантически отличается от цикла | --
| Цикл vs дубликат | Цикл: `isCycle=true` (координата в предках текущего пути). Дубликат: `isDuplicate=true` (координата уже резолвлена в другой ветке). Разные флаги для различения | --
| Глубина > maxDepth (default=10) | Резолвинг останавливается, ветка обрезается, WARNING. Настраивается через `settings.transitiveResolution.maxDepth` | `TRANSITIVE_MAX_DEPTH`
| > 5000 транзитивных | Post-processing: WARNING при >5000 узлов (nodeCounter) | `TRANSITIVE_LARGE_TREE`
| > 10000 транзитивных (maxTransitives) | Preemptive: nodeCounter проверяется в начале `resolveTree()`, early return при превышении. Post-processing: ERROR. Настраивается через `settings.transitiveResolution.maxTransitives` (default=10000) | `TRANSITIVE_MAX_EXCEEDED`
| Constraint на несуществующую библиотеку | WARNING "Constraint does not match" | `TRANSITIVE_UNUSED_CONSTRAINT`
| Конфликтующие constraints | ERROR "Conflicting constraints" | `TRANSITIVE_CONFLICTING_CONSTRAINTS`
| SNAPSHOT версии | Резолвятся как обычные, WARNING о нестабильности | `TRANSITIVE_SNAPSHOT`
| Optional dependencies | Не включаются по умолчанию, опция includeOptional | --
| Provided/Test scope | Не включаются по умолчанию, опция scopes | --
| Parent POM | Parent POM резолвится, версии и dependencies наследуются | --
| BOM в dependencyManagement | BOM резолвится, managed versions применяются | --
| Пустой каталог | 0 транзитивных, INFO | `TRANSITIVE_NO_LIBS`
| Таймаут при скачивании POM | Retry 3 раза, затем WARNING, пустое дерево | `TRANSITIVE_TIMEOUT`
| Дубликаты в дереве | Помечаются `isDuplicate=true` через sessionSeen, children=emptyList() для дубликатов, в flat list -- одна запись. Циклы помечаются `isCycle=true` (отдельный флаг) | --
| Параллельный доступ к sessionSeen | ConcurrentHashMap.newKeySet() -- thread-safe | --
| Кеш устарел (TTL) | Перезапрос из repository | --
| Сеть недоступна, кеш есть | Используется стейл кеш + WARNING | `TRANSITIVE_STALE_CACHE`
| Параллельный резолвинг | Общие зависимости резолвятся один раз через кеш | --
| Прерывание (Ctrl+C) | Graceful shutdown, частичные данные не сохраняются | --
|===

| `stale cache on network failure` | Сеть недоступна, кеш есть -> кеш + WARNING

[[priority]]
== Приоритет

**P4** (будущее) -- резолвинг транзитивных зависимостей предоставляет полную видимость для security и license проверок.
Включается через `withTransitiveResolution()` или `processing { resolveTransitives = true }`.
